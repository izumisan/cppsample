# C++17 プラクティス

C++17の新機能を使ってみた.

# overview

- InlineVariable
    - インライン変数
- StructuredBindings
    - 構造化束縛
- EnumClassInitializer
    - `enum class`の初期化時の制限緩和
- NewFunctions
    - c++17で追加された関数のサンプル
        - `for_each_n`
        - `clamp`
        - `hypot`
        - `gcd`
        - `lcm`
- AnyClass
    - anyクラス
- Filesystem
    - `<filesystem>`ヘッダ
        - `path`クラス
        - `directory_entry`クラス
        - `directory_iterator`クラス
